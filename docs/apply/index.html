Crear — Extensión y experimentación

En la etapa de Crear el estudiante o investigador deja de ser un usuario del código para convertirse en su autor. Aquí se integran los conocimientos adquiridos durante las etapas previas y se utilizan para extender o modificar la librería, probando nuevas ideas y algoritmos.

Crear es transformar la comprensión en acción. Cada nueva línea de código es un experimento, una hipótesis puesta a prueba.

Este proyecto está pensado para ser libre y extensible. Puedes descargar o clonar el repositorio en GitHub: 

git clone https://github.com/recore799/schrodinger1d.git
cd schrodinger1d

Si deseas colaborar oficialmente, solicita acceso o realiza un fork de la versión pública. Las contribuciones pueden centrarse en mejorar la eficiencia del algoritmo, ampliar la base de orbitales o añadir nuevos ejemplos visuales.

Ejemplos de creación

A continuación se muestran fragmentos de código extraídos de los scripts de examples/. Cada uno ilustra cómo generar nuevas visualizaciones o extender los modelos existentes.

1. Visualizar funciones de onda del oscilador armónico

import numpy as np
import matplotlib.pyplot as plt
from src.numerov.numerov import solve_oscillator

n = 3  # Nivel de energía
x, psi = solve_oscillator(n)

plt.plot(x, psi**2)
plt.title(f"Densidad de probabilidad para n={n}")
plt.xlabel("x")
plt.ylabel("|ψ|²")
plt.show()

2. Modificar el potencial en el átomo de hidrógeno

from src.numerov.numerov import solve_atom

Z = 1
def potential(r):
    return -Z / r + 0.05 * r  # Pequeña perturbación lineal

r, psi = solve_atom(n=2, l=0, potential=potential)
plt.plot(r, psi)
plt.title("Estado perturbado del átomo de hidrógeno")
plt.xlabel("r")
plt.ylabel("ψ(r)")
plt.show()

3. Extender el módulo de Hartree–Fock

Objetivos de la etapa

  • Aprender a modificar y extender la base de código existente.
  • Explorar el impacto físico de los cambios en los potenciales o integrales.
  • Generar nuevas visualizaciones y compararlas con resultados teóricos.
  • Desarrollar una comprensión práctica del proceso de investigación computacional.
