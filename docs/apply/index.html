Aplicar — Visualización y experimentos

En la etapa de Aplicar, los conocimientos adquiridos se utilizan de manera práctica: es el punto donde las ideas se transforman en resultados visibles. Aquí el estudiante ejecuta los métodos, ajusta parámetros y genera datos para observar la conexión entre teoría y experiencia numérica.

Aplicar es hacer operativa la comprensión: usar lo aprendido para resolver, modelar o visualizar un fenómeno físico.

Los ejemplos siguientes muestran cómo utilizar las rutinas del proyecto para representar funciones de onda, densidades electrónicas y comportamientos de sistemas cuánticos simples.

Ejemplo 1 — Funciones de onda del oscilador armónico

Se visualizan los estados estacionarios calculados con el método de Numerov, mostrando las densidades de probabilidad para distintos niveles cuánticos. 

import numpy as np
import matplotlib.pyplot as plt
from src.numerov.numerov import solve_oscillator

levels = [0, 1, 2, 3]
for n in levels:
    x, psi = solve_oscillator(n)
    plt.plot(x, psi**2, label=f"n = {n}")

plt.title("Densidad de probabilidad |ψ(x)|² del oscilador armónico")
plt.xlabel("x")
plt.ylabel("|ψ|²")
plt.legend()
plt.show()

Ejemplo 2 — Densidad electrónica del átomo de hidrógeno

A partir de las funciones radiales \(R_{nl}(r)\), se obtiene la densidad \(\rho(r) = |R_{nl}(r)|^2 r^2\) para distintos estados del átomo de hidrógeno. 

from src.numerov.numerov import solve_atom
import matplotlib.pyplot as plt
import numpy as np

states = [(1,0), (2,0), (2,1)]
for n, l in states:
    r, psi = solve_atom(n=n, l=l)
    rho = psi**2 * r**2
    plt.plot(r, rho, label=f"n={n}, l={l}")

plt.title("Densidad radial del átomo de hidrógeno")
plt.xlabel("r [a.u.]")
plt.ylabel("ρ(r)")
plt.legend()
plt.show()

Ejemplo 3 — Mapa 2D de densidad cuántica

Se genera un mapa bidimensional de la densidad de probabilidad para un estado del átomo de hidrógeno, combinando las soluciones radiales y armónicos esféricos. 

import numpy as np
import matplotlib.pyplot as plt
from scipy.special import sph_harm
from src.numerov.numerov import solve_atom

n, l, m = 3, 1, 0
r, psi = solve_atom(n, l)

# Construir grilla 2D
theta = np.linspace(0, np.pi, 300)
phi = np.linspace(0, 2*np.pi, 300)
R, PHI = np.meshgrid(r, phi)
THETA = np.linspace(0, np.pi, len(r))
Ylm = sph_harm(m, l, 0, THETA)
density = (abs(psi[:, None] * Ylm)**2).T

plt.imshow(density, extent=[0, max(r), 0, np.pi],
           aspect='auto', origin='lower', cmap='plasma')
plt.xlabel("r")
plt.ylabel("θ")
plt.title(f"Densidad cuántica para (n={n}, l={l}, m={m})")
plt.colorbar(label="|ψ|²")
plt.show()
