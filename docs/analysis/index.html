Analizar — Descomposición y estructura

En la etapa de Analizar se descompone el conocimiento adquirido para entender la estructura interna de los modelos y algoritmos utilizados. Aquí el estudiante examina las aproximaciones, dependencias y límites de los métodos aplicados en mecánica cuántica computacional.

Analizar es ver debajo de la superficie de la teoría: reconocer qué supuestos hacen posible un modelo y cómo esos supuestos determinan su alcance.

En esta sección se estudian componentes críticos como las bases atómicas (por ejemplo, la base STO-3G), la estabilidad del algoritmo de Hartree–Fock y el impacto de las aproximaciones numéricas.

Descomposición de la base STO-3G

La base STO-3G (Slater-Type Orbital con 3 Gaussianas) es una de las aproximaciones más empleadas para representar funciones de tipo Slater mediante combinaciones de gaussianas normalizadas. Su función objetivo es reproducir el decaimiento exponencial de los orbitales atómicos con una forma que permita evaluar las integrales analíticamente. 


φ_STO-3G(r) = c₁ e^{-α₁ r²} + c₂ e^{-α₂ r²} + c₃ e^{-α₃ r²}

Los coeficientes \(c_i\) y exponentes \(α_i\) se ajustan para aproximar la función de tipo Slater \(ψ(r) = N e^{-\zeta r}\). Cada orbital atómico (por ejemplo, 1s, 2p) se representa por una combinación lineal de tres gaussianas denominadas primitivas. La función resultante se denomina contracción.

  • Cada función base tiene tres primitivas gaussianas con pesos \(c_i\).
  • Es una base mínima: una función por orbital ocupado.
  • Las integrales de dos electrones se pueden calcular de forma analítica, lo que hace al método de Hartree–Fock computacionalmente viable.

El uso de funciones gaussianas en lugar de funciones Slater introduce una aproximación controlada. Aunque las funciones gaussianas no reproducen correctamente el comportamiento cuspado en el núcleo, su eficiencia en las integrales compensa esta limitación en cálculos de pequeño y mediano tamaño.

Analizar la construcción de la base STO-3G permite entender el equilibrio entre precisión física y viabilidad computacional.

Dependencia del método de Hartree–Fock con la base

Las propiedades y energías obtenidas mediante Hartree–Fock dependen fuertemente de la base elegida. A medida que se amplía el conjunto de funciones (por ejemplo, 6-31G o cc-pVDZ), las soluciones se acercan al límite de Hartree–Fock.

  • El tamaño de la base define la dimensionalidad del problema matricial \(FC = SC\epsilon\).
  • La elección de la base influye en la convergencia y estabilidad del procedimiento SCF.
  • Analizar diferentes bases permite estimar el error de truncamiento del espacio de Hilbert aproximado.
