Explorando sistemas cuánticos en Python

Una guía para el salón de clases — presentada en el Congreso Nacional de Física.

¿Por qué una pirámide?

Este proyecto aplica la Taxonomía de Bloom como modelo pedagógico para estructurar el aprendizaje computacional de la mecánica cuántica. Cada nivel de la pirámide representa un tipo de actividad de aprendizaje:

  • Recordar / Comprender: teoría básica y deducciones del oscilador armónico, el átomo de hidrógeno y Hartree–Fock.
  • Aplicar: simulaciones numéricas del método de Numerov y Hartree–Fock.
  • Analizar / Evaluar / Crear: exploración, discusión y extensión del código.

Haz clic en cada nivel de la pirámide para acceder a materiales teóricos, cuadernos interactivos y código experimental.

Haz clic en un nivel para abrir su contenido dentro de esta página.
