Explorando sistemas cuánticos en Python

Una guía para el salón de clases — presentada en el Congreso Nacional de Física.

¿Por qué una pirámide?

Este proyecto aplica la Taxonomía de Bloom como modelo pedagógico para estructurar el aprendizaje computacional de la mecánica cuántica. Cada nivel de la pirámide representa un tipo de actividad de aprendizaje:

  • Recordar / Comprender: teoría básica y deducciones del oscilador armónico.
  • Aplicar: simulaciones numéricas del método de Numerov.
  • Analizar / Evaluar / Crear: uso y extensión de algoritmos tipo Hartree-Fock en Python.

Haz clic en cada nivel de la pirámide para acceder a materiales teóricos, cuadernos interactivos y código experimental. Todo el contenido está desarrollado con software libre y puede adaptarse para cursos universitarios de física cuántica.

Todos los niveles abren el mismo PDF de prueba. Próximamente enlazaremos cada uno a su propio recurso.
